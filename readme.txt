== Description ==

This plugin add the "Lifetime/Onetime Purchase" functionality to Woocommerce Subscriptions plugin .

== Usage ==

Activate the plugin and you'll get an extra option ("Lifetime") in billing interval of subscription.

Optional:
Change text "$x.xx/Lifetime"  to "$x.xx/Your custom text". Goto Woocommerce -> Settings -> Subscriptions.
Then goto section "Lifetime Option Text".

== Installation ==

Installing "Woo Subscriptions Variation Lifetime/Onetime Purchase" can be done either by searching for "Woo Subscriptions Variation Lifetime/Onetime Purchase" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress
